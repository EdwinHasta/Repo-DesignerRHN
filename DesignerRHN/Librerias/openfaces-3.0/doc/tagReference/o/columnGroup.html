Tag columnGroup

The ColumnGroup component can be used to combine several columns into a group. The columns inside that are combined into a group can have an additional header and/or footer that spans across all of these columns. It is also possible to specify style and events for column groups. The set of attributes of tag is similar to that of tag. In order to create a group of columns, it is just needed to enclose the appropriate tags into a tag.
