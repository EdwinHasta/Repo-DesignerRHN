Tag commandLink

This is an analog of the standard component extended with Ajax features making it possible to use Ajax request instead of form submission to execute an action and reload components. By default it behaves as its standard analog though Ajax functionality will be turned on if you specify its "render" attribute, which is similar to that of component and specifies the list of components that should be reloaded when the link is clicked. An additional optional attribute is "execute", which just like in the component allows specifying a list of components whose data should be submitted to the server for processing during the Ajax request.
