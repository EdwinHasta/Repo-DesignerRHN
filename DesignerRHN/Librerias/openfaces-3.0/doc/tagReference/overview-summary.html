OpenFaces 3.0.0 Tag Library Documentation

The format of the OpenFaces tag library documentation has been modified to best suit JSF components tag library reference.
In particular, the following changes were made:
Removed columns from Attributes tables:
  • Request Time, since all attributes in the context of JSF have the same value - "false".
  • Type, since all attributes in the context of JSF have the same value of "java.lang.String".
Three new columns were added:
  • Binding Type - type of an object to which the attribute should be bound (if binding is supported).
  • EL - the flag that indicates whether or not the attribute supports binding. Can take the following values:
    • true - the attribute supports binding.
    • false - the attribute doesn't support binding.
    • only - the attribute must be specified as a value-binding expression.
  • Default - the default value of the attribute.
